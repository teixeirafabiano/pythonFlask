# pythonFlask

Criar Web APIs usando Python e Flask
====================================

<h1>Pré-requisitos</h1>
Python 3, <a href="https://flask.palletsprojects.com/en/1.1.x/">Flask Web Framework</a> e browser são necessários para esse "tutorial". Para quem não possui Python instalado, sugiro instalar pelo pacote do <a href="https://www.anaconda.com/">Anaconda</a>.
